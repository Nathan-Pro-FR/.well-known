# 📖 Guide des Commandes

```
/fetch tags:<mot_clé> [limit] [score] [ai] [spoiler]
```

| **Option**                                 | **Statut (Badge)**                                   | **Description**                                                                                                          |
| ------------------------------------------ | ---------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------ |
| <mark style="color:$danger;">`tags`</mark> | 🔴 <mark style="color:$danger;">`Obligatoire`</mark> | <p>Les mots-clés de ta recherche (ex: <code>hatsune_miku</code>). </p><p><del>Sépare les mots par des espaces.</del></p> |
| `limit`                                    | ⚪ `Optionnel`                                        | Nombre maximum d'images à recevoir en une seule fois (configurable de 1 à 10, par défaut `1`).                           |
| `score`                                    | ⚪ `Optionnel`                                        | Score minimum de qualité de l'image sur le site (par défaut `100` pour filtrer le contenu qualitatif).                   |
| `ai`                                       | ⚪ `Optionnel`                                        | Active-le (`true`) pour inclure les images générées par IA. Par défaut sur `false` (bloqué d'office).                    |
| `spoiler`                                  | ⚪ `Optionnel`                                        | Applique un flou "Spoiler" sur l'image dans Discord (par défaut sur `true` pour rester discret).                         |
